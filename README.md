# たばコン

公開URL: https://chameleonjp.codeberg.page/tabakon/

game_slug: `tabakon`

## 概要

「たばコン」は、砂を削って🚬を🦊へ届けるスマホ向けブラウザパズルゲームです。Sand Balls風の「掘って流れを作る」遊びを参考にしつつ、車両配送の構図は使わず、🚬を🦊の回収ゾーンへ届けるゲームとして実装しています。ステージは10種類からゲーム開始ごとにランダムで選ばれます。ステージ1「三つ道」を基準に、ステージ2〜10は左安全路、中央ゲート、右高得点路、二段バー、広い受け皿、細い通路、危険サンド、回転障害物、総合として、砂・壁・掘り抜き導線・危険ゾーン・ゴール前受け皿・バー/ゲート/ローターの配置を明確に差別化しています。

## 操作方法

- 名前を入力して「ゲーム開始」を押します。
- 3、2、1のカウント後にゲームが始まります。
- Canvas上を指でなぞると砂を削れます。
- 60秒以内に20個以上の🚬を🦊へ届けると成功です。
- ルール画面では色別🚬の点数（白っぽい🚬1000点、赤い🚬1500点、青い🚬1300点、黒い🚬1700点）と特徴を確認できます。
- 危険ゾーンに入った🚬は失われます。黒い🚬は危険ゾーンに近いだけでも失いやすいので注意します。危険ゾーンはCanvas内で赤黒または黒紫の塗り、赤枠、点滅、ラベル「危険」または「×」で表示します。
- リタイアボタンで途中終了できます。

## ランキング仕様

- スコア名: 配達スコア
- 単位: 点
- 順序: 高い方がよい `desc`
- 種別: ベストスコア型 `best`
- ステージ別ランキングは作らず、全10ステージ共通で `tabakon` の配達スコアランキングに送信
- 小数: なし
- 送信: クリア、失敗、リタイア、マイナススコアを含め、ゲーム終了時に自動で1回だけ送信
- Supabase URL: 共通設定 `https://mlpnjgezrnhdxsxolyzj.supabase.co`
- 送信: `submit_score` RPC（`/rest/v1/rpc/submit_score`）
- 取得: `get_best_score_ranking` RPC（`display_name` / `rank_no` 対応）
- 禁止: `/rest/v1/game_scores` への直接insert、旧ランキングテーブル、管理者用キー、サーバー用キー

## 公開方法

ビルドは不要です。リポジトリの `index.html` をそのまま Codeberg Pages で公開できる構成です。

## 注意事項

- ゲーム本体は `index.html` 1ファイルにHTML/CSS/JavaScriptをまとめています。
- npm、Vite、ビルド環境は不要です。
- 名前は `localStorage` の `tabakon_player_name` に保存しますが、公式ランキングのスコア本体は保存しません。
- SupabaseにはPublishable keyのみを記載しています。
- Canvasの48×72グリッドは固定セルサイズではなく、Canvas実サイズから算出した `CELL_W` / `CELL_H` で描画と判定を揃えます。
- ゲーム中と結果画面に選ばれたステージ名を表示します。ランキングはステージ別に分けず、全ステージ共通です。
- 結果画面には届けた点、残り時間、失ったペナルティ、リタイアペナルティ、合計のスコア内訳を表示します。
- RESULT後は `stopLoop()` で描画ループを停止します。

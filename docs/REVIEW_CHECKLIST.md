# たばコン レビュー検査リスト

## 仕様違反確認

- [ ] ゲーム名が「たばコン」。
- [ ] game_slug が `tabakon`。
- [ ] 公開URLが `https://chameleonjp.codeberg.page/tabakon/`。
- [ ] 🚬を🦊へ届ける内容。
- [ ] 車両配送表現を使っていない。
- [ ] 旧仕様の文字列や絵文字が残っていない。
- [ ] ステージは10種類からランダムで1つ選ばれる。
- [ ] ステージ1〜10の `name` / `shortName` がある。
- [ ] ステージ2〜10は砂、壁、掘り抜き導線、危険ゾーン、ゴール前受け皿、仕掛け配置が名前だけ違うコピーではなく、ギミック配置で差別化されている。
- [ ] ゲーム中と結果画面に選ばれたステージ名が表示される。
- [ ] ステージ別ランキングはなく、`tabakon` 共通ランキングのまま。

## スマホ表示確認

- [ ] iPhone SE幅で操作できる。
- [ ] 横スクロールが出ない。
- [ ] ボタンのタップ範囲が十分。
- [ ] 下部ボタンが隠れない。
- [ ] Canvasスワイプでページスクロールしない。
- [ ] Canvas表示サイズは320×480相当のままで、ページ全体を960px表示にしていない。
- [ ] 左端の縦スクロールバーで上端から下端まで `cameraY` を動かせる。

## Canvas描画確認

- [ ] 砂、🚬、🦊、壁、トラップ、動くバー、開閉ゲート、回転障害物をCanvasで描画。
- [ ] 48×144グリッドを固定セルサイズではなく `CELL_W` / `CELL_H` で320×960の世界座標に合わせて描画・判定している。
- [ ] 砂セル描画は表示範囲周辺に絞られている。
- [ ] スクロールバーに現在表示範囲、🦊位置、🚬平均位置の目印がある。
- [ ] 色の異なる🚬がある。
- [ ] ルール画面に色別🚬の点数と特徴が表示される。
- [ ] 危険ゾーンの説明がルール画面に表示される。
- [ ] 複雑な地形と分岐がある。
- [ ] 危険ゾーンが黒紫/赤黒、赤枠、ラベル「危険」または「×」、点滅で見分けやすい。
- [ ] 各ステージに20個到達を狙える安全導線とゴール前の受け皿がある。
- [ ] ステージ2、6は初心者寄り、ステージ1、3、5、10は標準、ステージ4、7、8、9はやや難しい目標になっている。
- [ ] 初見10〜18個、普通に掘れば20個、上手いと23〜28個を狙える範囲に近い。
- [ ] ほぼ何もしなくても20個届くステージ、どう掘っても20個届かないステージがない。
- [ ] ステージ3「中央ゲート」が0〜5個付近で詰まり続けない。
- [ ] ステージ9「回転障害物」が開始直後に大量到達しすぎず、初見15〜22個、上手いと25個以上を狙える。

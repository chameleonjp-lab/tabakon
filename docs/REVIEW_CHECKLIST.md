# たばコン レビュー検査リスト

## 仕様違反確認

- [ ] ゲーム名が「たばコン」。
- [ ] game_slug が `tabakon`。
- [ ] 公開URLが `https://chameleonjp.codeberg.page/tabakon/`。
- [ ] 🚬を🦊へ届ける内容。
- [ ] 車両配送表現を使っていない。
- [ ] 旧仕様の文字列や絵文字が残っていない。
- [ ] ステージは10種類からランダムで1つ選ばれる。
- [ ] ステージ別ランキングはなく、`tabakon` 共通ランキングのまま。

## スマホ表示確認

- [ ] iPhone SE幅で操作できる。
- [ ] 横スクロールが出ない。
- [ ] ボタンのタップ範囲が十分。
- [ ] 下部ボタンが隠れない。
- [ ] Canvasスワイプでページスクロールしない。

## Canvas描画確認

- [ ] 砂、🚬、🦊、壁、トラップ、動くバー、開閉ゲート、回転障害物をCanvasで描画。
- [ ] 48×72グリッドを固定セルサイズではなく `CELL_W` / `CELL_H` でCanvasに合わせて描画・判定している。
- [ ] 色の異なる🚬がある。
- [ ] ルール画面に色別🚬の点数と特徴が表示される。
- [ ] 危険ゾーンの説明がルール画面に表示される。
- [ ] 複雑な地形と分岐がある。
- [ ] 危険ゾーンが黒紫/赤黒、赤枠、ラベル、点滅で見分けやすい。
- [ ] 各ステージに20個到達を狙える安全導線とゴール前の受け皿がある。

## 操作確認

- [ ] READYで3、2、1のカウントが出る。
- [ ] PLAYING中だけ砂を削れる。
- [ ] HOME、RULE、READY、RESULT、ERRORでゲーム処理が進まない。
- [ ] リタイアできる。

## 結果画面確認

- [ ] 成功、失敗、リタイアを表示。
- [ ] 配達スコア、届けた数、失った数、残り時間、遊んだステージ名を表示。
- [ ] 結果画面にスコア内訳を表示。

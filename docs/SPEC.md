# たばコン 仕様書

## 基本情報

- ゲーム名: たばコン
- game_slug: `tabakon`
- 公開URL: `https://chameleonjp.codeberg.page/tabakon/`
- 概要: 砂を削って、🚬を🦊へ届けるスマホ向けパズルゲーム。
- 形式: `index.html` 1ファイルでゲーム本体を完結。

## ゲーム目的

Canvas上の砂を指で削り、30個の🚬を重力で流して🦊の回収ゾーンへ届ける。20個以上届けると成功。

## 状態

- HOME
- RULE
- READY
- PLAYING
- RESULT
- ERROR

READYでは3、2、1のカウントのみ行い、PLAYING中だけ砂削り、🚬移動、仕掛け更新、当たり判定、スコア加算を行う。RESULT後は `stopLoop()` で描画ループを含むゲーム処理を停止する。

## ステージ

Canvasは320×480表示、砂グリッドは48×72で管理する。セル座標とCanvas座標の変換は固定値ではなく、Canvas実サイズから算出した `CELL_W` / `CELL_H` を使って描画、砂削り、当たり判定のずれを防ぐ。

ステージは `STAGES` 配列で10種類を定義し、ゲーム開始またはリトライごとにランダムで1つだけ選ぶ。ステージ別ランキングは作らず、全ステージ共通の `tabakon` 配達スコアランキングを使う。各ステージには20個到達を狙える安全導線、危険または高得点ルート、危険ゾーンを避けられるルート、ゴール前の受け皿を用意する。ゲーム中と結果画面には選ばれたステージ名を表示する。

ステージ1「三つ道」は基準ステージとして残し、ステージ2〜10はギミック配置で差別化する。ステージ2「左安全路」は左安全ルートと右危険ルート、ステージ3「中央ゲート」は中央の開閉ゲート、ステージ4「右高得点路」は右側の高得点/危険ルート、ステージ5「二段バー」は上下2段の移動バー、ステージ6「広い受け皿」は届きやすい広い下層、ステージ7「細い通路」は精度が必要なゴール前細道、ステージ8「危険サンド」は多めの危険ゾーンと安全導線、ステージ9「回転障害物」はローター、ステージ10「総合」は左右分岐・ゲート・バー・ローターを組み合わせる。

難易度目標は、初見でも10〜18個程度が届き、普通に掘れば20個成功を狙え、上手いプレイでは23〜28個を狙える範囲に揃える。ステージ2、6は初心者寄り、ステージ1、3、5、10は標準、ステージ4、7、8、9はやや難しい位置づけにする。ただし、初心者寄りでも放置だけで20個届く状態は避け、やや難しいステージでも20個到達導線を必ず残す。ステージ3は中央ゲートへの導線とゲート下の受け皿を広めにし、0〜5個付近で詰まり続けないようにする。ステージ9は中央落下路を少し絞り、ローターと危険ゾーンでタイミング調整を作りつつ、気持ちよく流れる余地を残す。

## 🚬の種類

| 種類 | 点数 | 特徴 |
| --- | ---: | --- |
| 白っぽい🚬 | 1000 | 標準の動き。まずはこれを🦊へ届ける。 |
| 赤い🚬 | 1500 | 重くて落ちやすいが、横に転がりにくい。 |
| 青い🚬 | 1300 | 軽くて跳ねやすい。バーや回転障害物で動きやすい。 |
| 黒い🚬 | 1700 | 高得点。ただし危険ゾーンに近づくと失いやすい。 |

## 仕掛けとトラップ

- 左右に動くバー、開閉するゲート、回転する障害物をステージごとに配置する。仕掛けは最大5個。
- トラップ: 最大6か所。落とし穴、危険エリア、黒い沼風エリア。危険ゾーンは黒紫または赤黒の塗り、赤い枠線、「危険」または「×」のCanvas内ラベル、軽い点滅で砂や壁と見分けやすく描画する。
- ルール画面には「危険ゾーンに🚬が入ると失います。黒い🚬は危険ゾーンに近いだけでも失いやすいので注意。」を表示する。

## 終了条件

PLAYING中に20個以上届けると成功演出後に結果へ進む。制限時間60秒が0になった場合は、届けた数が20個以上なら成功、20個未満なら失敗として結果へ進む。

全🚬が届けた、失った、または画面外になり `active=false` になった場合は、制限時間を待たずに結果へ進む。さらに、アクティブな🚬が残っていても、その全件がゴール範囲より下に落ちてもう届かない位置にある場合は、即座に結果へ進む。この即終了でも、20個以上届けていれば成功、20個未満なら失敗とし、結果画面、スコア送信、ランキング送信は通常終了と同じく1回だけ行う。リタイア処理は従来通りプレイヤー操作による終了として扱う。

## スコア

配達スコア = 届けた🚬の点数合計 + 残り時間ボーナス - 失った🚬の数×300 - リタイアペナルティ。結果画面にはステージ名とスコア内訳を表示する。

- 残り時間ボーナス: 残りミリ秒 ÷ 10 の整数化
- リタイアペナルティ: 5000
- マイナススコア許可
- 失敗、リタイア、マイナススコアもランキング送信

## スマホ対応

Canvasに `touch-action: none` を設定し、横スクロール、長押し選択、ダブルタップ拡大、ピンチズームによる崩れを抑える。iPhone SE幅でも遊べる縦画面設計とする。

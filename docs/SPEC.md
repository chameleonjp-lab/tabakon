# たばコン 仕様書

## 基本情報

- ゲーム名: たばコン
- game_slug: `tabakon`
- 公開URL: `https://chameleonjp.codeberg.page/tabakon/`
- 概要: 砂を削って、🚬を🦊へ届けるスマホ向けパズルゲーム。
- 形式: `index.html` 1ファイルでゲーム本体を完結。

## ゲーム目的

Canvas上の砂を指で削り、30個の🚬を重力で流して🦊の回収ゾーンへ届ける。20個以上届けると成功。

## 状態

- HOME
- RULE
- READY
- PLAYING
- RESULT
- ERROR

READYでは3、2、1のカウントのみ行い、PLAYING中だけ砂削り、🚬移動、仕掛け更新、当たり判定、スコア加算を行う。RESULT後は `stopLoop()` で描画ループを含むゲーム処理を停止する。

## ステージ

Canvasは320×480表示、砂グリッドは48×72で管理する。セル座標とCanvas座標の変換は固定値ではなく、Canvas実サイズから算出した `CELL_W` / `CELL_H` を使って描画、砂削り、当たり判定のずれを防ぐ。

1ステージ制。上部に🚬のまとまり、中央に厚い砂山、左右分岐、片側トラップ、片側遠回り安全通路、下部に🦊回収ゾーン、ゴール手前の狭い通路、中央の動くバーを配置する。

## 🚬の種類

| 種類 | 点数 | 特徴 |
| --- | ---: | --- |
| 通常🚬 | 1000 | 標準の動き |
| 赤い🚬 | 1500 | 少し重く転がりにくい |
| 青い🚬 | 1300 | 少し跳ねやすく速い |
| 黒い🚬 | 1700 | トラップに近づくと消えやすい |

## 仕掛けとトラップ

- 左右に動くバー: 最大3個
- 開閉するゲート: 最大1個
- 回転する障害物: 最大1個
- トラップ: 最大6か所。落とし穴、危険エリア、黒い沼風エリア。

## スコア

配達スコア = 届けた🚬の点数合計 + 残り時間ボーナス - 失った🚬の数×300 - リタイアペナルティ。

- 残り時間ボーナス: 残りミリ秒 ÷ 10 の整数化
- リタイアペナルティ: 5000
- マイナススコア許可
- 失敗、リタイア、マイナススコアもランキング送信

## ランキング

- Supabase URL: 共通設定 `https://mlpnjgezrnhdxsxolyzj.supabase.co`
- 送信RPC: `submit_score`（`/rest/v1/rpc/submit_score`）
- 取得RPC: `get_best_score_ranking`
- 表示: `display_name`、`player_name`、`name` の順で名前を参照し、`rank_no` / `rank` に対応する
- 禁止: `/rest/v1/game_scores` への直接insert、旧ランキングテーブル
- key: Publishable keyのみ
- 送信: 終了時に自動で1回だけ
- 結果画面にランキング登録ボタンは置かない

## スマホ対応

Canvasに `touch-action: none` を設定し、横スクロール、長押し選択、ダブルタップ拡大、ピンチズームによる崩れを抑える。iPhone SE幅でも遊べる縦画面設計とする。
